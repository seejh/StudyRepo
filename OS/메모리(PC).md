
# 메모리 종류
DRAM, SRAM, ROM<br/>

* DRAM(Dynamic Random Access Memory)<br/>
  전원이 꺼지면 사라지는 휘발성 메모리.
* SRAM(Static Random Access Memroy)<br/>
  DRAM보다 빠르고 안정적이나 가격이 비싸서 주로 캐시로 사용.
* ROM(Read Only Memory)<br/>
  데이터를 읽기만 가능한 메모리로, 컴퓨터의 기본 설정 정보 저장에 사용.

# 용량
컴퓨터가 작업하고 있는 정보들을 임시로 저장하고 있는 공간 크기.

# CL(CAS Latency)
Column Address Strobe Latency.<br/>
메모리가 특정 데이터가 있는 열(Column)을 찾은 후 실제로 데이터를 전달하기까지 걸리는 클럭 수를 의미.<br/>
"지연 시간, 반응 속도"라고 생각하면 되며 낮을수록 좋다.<br/>

CL16 = 16클럭 동안 기다려야 데이터가 나온다.<br/>

# 클럭
메모리가 "1초에 몇 번 데이터를 읽고 쓸 수 있는지"를 나타내며 MHz(메가헤르츠) 단위로 표현.<br/>
초당 어느 정도의 데이터를 처리할 수 있는지로, 대역폭을 나타내며 높을수록 좋고 한 번에 처리할 수 있는 데이터량이
높다.

# 메모리 채널
CPU의 메모리 채널은 물리적으로 64bit 데이터 버스 1개 => 대역폭 64bit.<br/>
2개 사용할 경우 128bit.<br/>

i7-8700k 기준, 2개의 메모리 채널 가용 가능.<br/>
듀얼 채널로 구성 시에 128bit 대역폭을 가짐.<br/>

# XMP, EXPO
메모리 오버클록 프로필 기술.<br/>
제조사가 최고 성능 설정을 미리 프로필에 저장하고, 쉽게 불러와서 오버클록을 적용하는 기술.<br/>
XMP(인텔), EXPO(AMD).<br/>

적용 방법은 BIOS 진입 -> 고급 모드 전환 -> XMP 활성화 -> 저장.<br/>
그러니까 메인보드와 램에서 둘 다 이 기술을 지원해야 한다.
