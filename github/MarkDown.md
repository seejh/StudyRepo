마크다운이란?<br/>
마크업 언어의 일종이다. 마크업 언어란, 텍스트에 태그를 추가하는 방식으로 문서의 형식을 정의하는 언어를 말한다.
주로 웹 페이지의 구조와 디자인을 결정하는데 사용.
