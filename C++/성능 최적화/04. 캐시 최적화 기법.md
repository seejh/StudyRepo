캐시 최적화의 중요성<br/>
캐시 메모리는 CPU가 메인 메모리보다 훨씬 빠르게 접근할 수 있는 작은 크기의 고속 메모리이다. <br/>
CPU 캐시를 최대한 활용할 수 있도록 데이터를 배치해 메모리 접근 속도를 높인다.<br/>

데이터 로컬리티<br/>
데이터 로컬리티는 CPU가 데이터를 빠르게 접근할 수 있도록 데이터를 메모리에 배치하는 방법을 의미한다.<br/>
데이터 로컬리티에는 두 가지 유형이 있다.<br/>
