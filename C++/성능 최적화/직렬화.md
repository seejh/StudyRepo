# 직렬화
메모리 속에 있는 데이터나 객체를 저장하거나 네트워크로 전송할 수 있는 데이터 형식(바이트 스트림)으로 변환하는 것.

### 무슨 말? 왜 필요?
예를 들어 c++에서..<br/>
```
struct Player {
  int id;
  string name;
};
```
위와 같은 객체가 있으면 메모리에는 id 값과 string이 가진 포인터, 길이 정보 등이 특정한 방식으로 저장된다.
하지만 이 메모리 레이아웃은 플랫폼, 언어 별로 다를 것이고 포인터(주소)도 네트워크로 받는 곳과 다를 것이다.<br/>

그래서 네트워크 기준으로 표현하면 받는 곳과 보내는 곳이 어떤 약속한 규칙(포맷)에 맞게 데이터를 변환하여 보내고 받는 것.
