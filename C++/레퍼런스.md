# 1. 레퍼런스 반환
일반 변수 반환하는 것과 레퍼런스로 반환하는 것의 차이

```c++
int t1(); // 값 반환
int& t2(); // 레퍼런스 반환

// 값 반환
int x = 10;
int t1() { return x; }

t1() = 20; // 컴파일 에러

// 레퍼런스 반환
int x = 10;
int& t2() { return x; }

t2() = 20; // OK
```
* 값 반환
  * f() = 임시값(rvalue)
  * 수정할 수 있는 대상 아님
 
* 레퍼런스 반환
  * t2()는 x의 다른 이름(=별칭, alias)
  * 좌변(lvalue)로 사용 가능
