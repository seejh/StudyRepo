# 캐시
캐시 메모리는 저장 장치 간의 속도 차에 의한 병목 현상을 줄이기 위한 메모리로 프로세서 가까이에 위치하며
빈번하게 사용되는 데이터들을 놔두는 장소이다.



## 캐싱 라인


3가지 방식
### Direct Map 직접 매핑
메인메모리를 일정한 크기의 블록으로 나누어 각각의 블록을 캐시의 정해진 위치에 매핑하는 방식.
가장 간단, 구현 쉬움
다만 적중률이 낮아질 수 있고 동일한 캐시 메모리에 할당된 여러 데이터를 사용할 때 충돌이 발생하게 된다는 단점이 있다.
