# 네임스페이스 (namespace)
### 사용 목적
이름 충돌 방지, 코드 소속감 부여

### 사용 예제
```
// 선언(Test.h)
namespace test {
  void TestFunc1(); // 선언
  void TestFunc2();
}
// 정의(Test.cpp)
// .cpp의 전역을 감싸던가
namespace test {
  void TestFunc1() { cout << "test namespace Func1()" } // 정의
  void TestFunc2() { cout << "test namespace Func2()" }
}
// 하나씩 처리하던가
void test::TestFunc1() { cout << "test namespace Func1" << endl; }
void test::TestFunc2() { cout << "test namespace Func2" << endl; }

// 사용(main.cpp)
int main() {
  // 네임스페이스::함수명
  test::TestFunc1();

  // using으로 선언하면 함수명 그냥 사용 가능
  using namespace test;
  TestFunc1();
}
```

### 주의점
네임스페이스를 using으로 사용하는 경우...
```
// test 네임스페이스의 void Func()
namespace test { void TestFunc() { cout << "test namespace Func()" << endl; } }

// 글로벌 네임스페이스의 void Func()
void TestFunc() { cout << "global namespace Func()" << endl; }

using namespace test;
int main() {
  TestFunc(); // 이름 충돌 에러
  ::TestFunc(); // 글로벌 네임스페이스의 함수 호출
  test::TestFunc(); // test 네임스페이스의 함수 호출
}
```
