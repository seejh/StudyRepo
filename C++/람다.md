# 람다, 람다 함수

람다는 내부적으로 컴파일러가 생성한 익명 클래스(클로저 타입)로 변환된다.

* 클로저 타입?
  * 고유 타입으로 이름이 없고 auto, callable로 담아서 접근할 수 있다.
 
```
SomeClass obj = SomeClass {};
auto lambda = [&obj]() { obj.DoSomething(); }

이 위의 문장을 컴파일러는 아래와 같이 변환한다.

struct lambda {
  //
  lambda(SomeClass& obj) : obj(obj) {}

  //
  void operator()() const {
    obj.DoSomething();
  }

  //
  SomeClass& obj;
};

```
람다 함수는 클로저 타입의 클래스이고 람다 함수 객체 자체는 이 클로저 클래스의 인스턴스라고 보면 된다.

캡처된 변수들은 이 클로저 클래스의 멤버 변수, 함수가 되고 & 캡처는 참조 멤버, 값(=) 캡처는 복사 멤버가 된다.

& 참조를 사용할 경우 dangling에 주의해야 한다.
