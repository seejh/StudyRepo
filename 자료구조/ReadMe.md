[ 자료구조 (Data Structure) ]
│
├─ 선형 구조 (Linear ADT)
│   │
│   ├─ 리스트 (List)
│   │     ├─ (구현) 순차 리스트 (배열, vector)
│   │     └─ (구현) 연결 리스트 (singly, doubly)
│   │
│   ├─ 스택 (Stack)
│   │     └─ LIFO
│   │
│   ├─ 큐 (Queue)
│   │     └─ FIFO
│   │
│   └─ 덱 (Deque)
│         └─ 양쪽 삽입/삭제
│
└─ 비선형 구조 (Non-Linear ADT)
    │
    ├─ 트리 (Tree)
    │     │
    │     ├─ 일반 트리
    │     ├─ 이진 트리
    │     ├─ 이진 탐색 트리 (BST)
    │     ├─ 힙 (Heap)
    │     └─ 균형 트리 (AVL, Red-Black)
    │
    └─ 그래프 (Graph)
          │
          ├─ 방향 / 무방향
          ├─ 가중치 / 비가중치
          └─ 희소 / 밀집
