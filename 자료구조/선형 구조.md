# 선형 구조(Linear Data Structure)
데이터가 일렬로 나열, 각 원소는 최대 하나의 이전/다음 원소만 가진다.
```
A -> B -> C -> D

선형 구조 (Linear ADT)
│
├─ 리스트 (List)
├─ 스택 (Stack)
├─ 큐 (Queue)
│    ├─ 일반 큐
│    ├─ 원형 큐
│    └─ 우선순위 큐
└─ 덱 (Deque)
```

* 순서 있음
* 앞뒤 관계 명확
* 계층 구조 없음
